# BTSPD
PHP的在线排队叫号系统

#安装说明
第一步、解压 
第二步、配置/config下的配置文件 其中bts_cf_mysqli.php为mysql配置文件 btspd_cf.php为您的餐馆信息
第三步、除btspd.sql以外上传全部文件
第四步、创建数据库并导入btspd.sql
第五步、用文本编辑器替换菜品名称，替换菜品图片。
